# Marketing-Management

Wählen Sie das _Marketing-_-Symbol (Megaphon) in der linken Navigationsleiste, um Ihren zentralen Arbeitsbereich für die Planung, Organisation und Aktivierung von Marketing zu öffnen. Von hier aus können Sie [Programme](./programs.md), [Journey](./person-journeys.md) erstellen, [Personenlisten](../audiences/people-lists.md) verwalten und [Inhalte](../content/digital-asset-management.md) - alles an einem Ort.

Die Marketing-Verwaltung nutzt ein Layout mit drei Regionen: einen Chatbereich auf der linken Seite, einen Arbeitsbereich in der Mitte und einen Programmbaum auf der rechten Seite.

![Arbeitsbereich Marketing-Verwaltung](./assets/marketing-home-page.png){width="800" zoomable="yes"}

## Chat-Bedienfeld {#chat-panel}

Das Chat-Bedienfeld wird parallel zu Ihrer Arbeit ausgeführt, sodass Sie den [KI-](../agents/chat-interface.md)) bitten können, Dinge im Kontext zu tun. Die Panel-Kopfzeile enthält die folgenden Steuerelemente:

| Kontrollvariante | Beschreibung |
|---------|-------------|
| **Neue Konversation** | Beginne eine neue Unterhaltung. |
| **Konversationsverlauf** | Vergangene Unterhaltungen durchsuchen und erneut öffnen. |
| **Bedienfelder wechseln** | Schalte das Chat-Panel auf die andere Seite. |
| **Reduzieren** | Blenden Sie das Bedienfeld aus, um den Arbeitsbereich zu maximieren. |

Die Eingabe lautet _Alles fragen, oder / für Kenntnisse eingeben…_ Wenn Sie ein Asset im Arbeitsbereich ausgewählt haben, wird die Eingabe kontextabhängig (z. B. _Fragen Sie nach &quot;[Asset-Name]&quot;…_), sodass Ihre Fragen direkt auf das zutreffen, was Sie anzeigen.

## Arbeitsbereich {#workspace}

Der mittlere Arbeitsbereich ist der Bereich, in dem Assets zur Anzeige und Bearbeitung geöffnet sind.

Wenn nichts ausgewählt ist, zeigt der Arbeitsbereich einen Willkommensstatus an - **Willkommen zurück. Umsätze generieren sich nicht von selbst.** - mit einer Eingabeaufforderung zur Auswahl eines Ordners oder Assets aus der Programmstruktur.

- **Kürzlich aufgerufen** - Eine Liste der [Journey](./person-journeys.md), [Programme](./programs.md) und Ordner, die Sie zuletzt berührt haben, jeweils mit einem relativen Zeitstempel. Klicken Sie auf eine beliebige Zeile, um sie erneut zu öffnen.
- **Asset-Ansicht** - Sobald Sie etwas aus der Programmstruktur auswählen, werden seine Details im Arbeitsbereich geöffnet.
- **Inventarseiten** - Wenn Sie einen Ordner oder ein Programm auswählen, wird eine Inventartabelle mit dessen Inhalt angezeigt ([Programme](./programs.md), [Journey](./person-journeys.md), [Personenlisten](../audiences/people-lists.md), E-Mails usw.). Inventartabellen unterstützen die Spaltenanpassung, das Ein-/Ausblenden von Spalten, die Sortierung und die Suche. Wenn Sie beispielsweise eine Personenliste auswählen, wird das zugehörige Mitgliederinventar geöffnet.

## Programmbaum {#program-tree}

Im rechten Bedienfeld wird der Navigationsbaum für Ihre Marketing-bezogenen Assets angezeigt. Es bietet funktionale Tools, die Sie verwenden können:

- Ein **Programm erstellen**-Button oben (siehe [Programme](./programs.md)).
- Ein **Suchen**-Feld zum Suchen von Assets nach Namen.
- Eine hierarchische Ordnerstruktur, die auf _Marketing-Aktivitäten/Standard_ basiert und Ordner, [Programme](./programs.md) und [Journey &#x200B;](./person-journeys.md). Erweitern Sie Ordner, um einen Drill-in durchzuführen. Das Menü **…** in jeder Zeile zeigt Asset-spezifische Aktionen.

Wählen Sie ein Element im Programmbaum aus, um die Details im mittleren Arbeitsbereich zu öffnen.
