# Personen-Journey-Knoten

Nachdem Sie eine Personen-Journey erstellt haben, verwenden Sie die Tools auf der Arbeitsfläche, um die Audience hinzuzufügen und die Journey mithilfe von Knoten zu erstellen. Die Journey-Arbeitsfläche bietet einen visuellen Design-Bereich, in dem Sie Ihre mehrstufigen B2B-Marketing-Anwendungsfälle erstellen können.

Der Knoten _[!UICONTROL Person]_ Zielgruppe ist automatisch der erste Knoten auf der Journey. Nachdem Sie die Audience ausgewählt haben, erstellen Sie Ihren Journey, indem Sie die verschiedenen Aktions-, Ereignis- und Entscheidungsknoten als mehrstufiges, kanalübergreifendes Szenario kombinieren. Jeder Knoten einer Journey stellt einen Schritt entlang eines logischen Pfads dar.

![Zielgruppenknoten Person](https://cdn.experienceleague.adobe.com/icons/users.svg?lang=de)

Zielgruppenknoten Person

Die Journey beginnt immer mit dem Zielgruppenknoten Person . Legen Sie die Eingabe-Audience mithilfe einer statischen oder dynamischen Personenliste fest, um zu definieren, welche Personenprofile in die Journey eintreten.

[Informationen zum Zielgruppenknoten für Personen](./person-audience-node.md)
:::

:::
![Aktionsknoten](https://cdn.experienceleague.adobe.com/icons/circle-play.svg?lang=de)

Aktionsknoten

Wenden Sie Änderungen auf alle Personen in einem Pfad an, z. B. beim Senden einer E-Mail oder beim Aktualisieren eines Datenwertattributs.

[Informationen zu Aktionsknoten](./action-nodes.md)
:::

:::
![Ereignisknoten](https://cdn.experienceleague.adobe.com/icons/puzzle-piece.svg?lang=de)

Ereignisknoten

Fahren Sie mit dem nächsten Journey-Schritt fort, wenn eine bestimmte Aktivität oder Datenänderung auftritt, z. B. beim Öffnen einer E-Mail oder bei einer Datenwertänderung.

[Informationen zu Ereignisknoten](./listen-for-event-nodes.md)
:::

:::
![Knoten warten](https://cdn.experienceleague.adobe.com/icons/clock.svg?lang=de)

Warteknoten

Den Journey-Fortschritt für eine bestimmte Dauer oder bis zu einem bestimmten Datum und einer bestimmten Uhrzeit anhalten, bevor Personen zum nächsten Schritt bewegt werden.

[Informationen zu Warteknoten](./wait-nodes.md)
:::

:::
![Pfade aufteilen und zusammenführen](https://cdn.experienceleague.adobe.com/icons/code-branch.svg?lang=de)

Aufspalten und Zusammenführen von Pfadknoten

Personen basierend auf Bedingungen in separate Pfade unterteilen, unterschiedliche Aktionen oder Ereignisse pro Pfad anwenden und dann die Pfade zusammenführen, um das Journey fortzusetzen.

[Erfahren Sie mehr über Aufspaltungs- und Zusammenführungsknoten](./split-merge-paths-nodes.md)
:::

:::
![Nächster bester Pfadknoten](https://cdn.experienceleague.adobe.com/icons/lightbulb.svg?lang=de)

Nächster bester Pfadknoten

Verwenden Sie KI-gesteuerte Entscheidungsfindung , um jede Person zum relevantesten Pfad zu leiten. Beschreiben Sie Ihre Absicht in natürlicher Sprache und lassen Sie das System den Profilkontext, den Interaktionsverlauf und die Kontodaten auswerten, um die Routing-Entscheidung zu treffen.

[Erfahren Sie mehr über den nächstbesten Pfadknoten](./next-best-path.md)
:::
