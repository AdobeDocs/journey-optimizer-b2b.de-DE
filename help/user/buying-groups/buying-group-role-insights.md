# Dashboard für Rolleneinblicke

Das Dashboard „Rolleneinblicke“ bietet Einblicke in die Entwicklung und das Engagement von Einkaufsgruppenrollen im Laufe der Zeit. Es hilft Marketing-Experten, Trends bei der Rollenakquise, Interaktionsmuster und die Art und Weise zu verstehen, wie jüngste Kampagnen die Interaktion über verschiedene Rollen innerhalb von Einkaufsgruppen hinweg fördern.

Um auf dieses Dashboard zuzugreifen, erweitern Sie **[!UICONTROL Konten]** im linken Navigationsbereich und wählen Sie dann **[!UICONTROL Einkaufsgruppen]**. Wählen Sie die Registerkarte **[!UICONTROL Rolleneinblicke]** aus.

![Role Insights-Dashboard](assets/roles-insights-dashboard.png){width="800" zoomable="yes"}

Das Dashboard umfasst drei Ansichten:

| Ansicht | Beschreibung |
| ---- | ----------- |
| [!UICONTROL Rollenakquise im Zeitverlauf] | Visualisiert die Anzahl der Mitglieder, die im Laufe der Zeit zu Einkaufsgruppen hinzugefügt wurden, segmentiert nach Rolle und gruppiert nach Monat. |
| [!UICONTROL Interaktion nach Rolle] | Zeigt die Gesamtzahl der Interaktionsaktivitäten (wie E-Mail-Öffnungen, Klicks und Formularübermittlungen) nach Käufergruppenrolle an. |
| [!UICONTROL Interaktion nach Rolle im Zeitverlauf] | Verfolgt im Laufe der Zeit Käufe von Interaktionsaktivitäten für Gruppen und zeigt monatliche Interaktionstrends nach Rolle an. |

## Filtern der Daten

Klicken Sie oben links auf _Filter_-Symbol ![Filtersymbol](../assets/do-not-localize/icon-filter.svg) ), um die angezeigten Daten mithilfe eines der folgenden Attribute zu filtern:

* **[!UICONTROL Rolle]** - Filtert die Daten nach einer oder mehreren ausgewählten Einkaufsgruppenrollen.
* **[!UICONTROL Lösungsinteresse]**: Filtert die Daten nach einem oder mehreren ausgewählten Lösungsinteressen, um sich auf bestimmte Produktlinien oder Angebote zu konzentrieren.
* **[!UICONTROL Datumsbereich]** - Beschränkt die Daten auf einen bestimmten Zeitraum (standardmäßig auf das vergangene Jahr).

![Dialog „Filter“ für Rolleneinblicke](./assets/roles-insights-filter-dialog.png){width="400"}

Wählen Sie für jedes Attribut die Werte aus, die Sie zum Filtern der Daten verwenden möchten, und klicken Sie auf **[!UICONTROL Anwenden]**.

## [!UICONTROL Rollenakquise im Zeitverlauf]

Wenn Marketing- und Vertriebsteams neue Mitarbeiter gewinnen, die Interaktion fördern und Daten anreichern, werden neue Mitglieder der Einkaufsgruppe qualifiziert. Dieser Bericht zeigt, wie sich diese Bemühungen in neu qualifizierten Einkaufsgruppenmitgliedern niederschlagen, aufgeschlüsselt nach Rolle. Jedes Mal, wenn eine Person einer Einkaufsgruppe hinzugefügt wird, wird diese Hinzufügung im Bericht unter der entsprechenden Rolle aufgezeichnet.

![Dashboard „Rolleneinblicke“ - Rollenakquise im Zeitverlauf](assets/roles-insights-role-aquisition-over-time.png){width="600" zoomable="yes"}

Jede Linie im Diagramm stellt eine Rolle dar. Bewegen Sie den Mauszeiger über einen Punkt auf der Linie, um die Details anzuzeigen, darunter:

* Akquirierungsdatum
* Anzahl der Akquisen

Um detailliertere Informationen anzuzeigen, klicken Sie auf das Menüsymbol **…** oben rechts.

## [!UICONTROL Interaktion nach Rolle]

Dieser Bericht fasst das Gesamtvolumen der Interaktionsaktivität durch den Kauf der Gruppenrolle über den ausgewählten Zeitraum zusammen. Verwenden Sie diesen Bericht, um einen Überblick darüber zu erhalten, wie die einzelnen Rollen als Reaktion auf Ihre Marketing-Maßnahmen interagieren.

![Dashboard „Rolleneinblicke“ - Interaktion nach Rolle (Diagramm](assets/roles-insights-engagement-by-role.png){width="500" zoomable="yes"}

Jeder Balken im Diagramm stellt eine Rolle dar. Bewegen Sie den Mauszeiger über eine Leiste, um die Details zur angezeigten Anzahl anzuzeigen, darunter:

* Rollenname
* Anzahl der Interaktionen

Um detailliertere Informationen anzuzeigen, klicken Sie auf das Menüsymbol **…** oben rechts.

## [!UICONTROL Interaktion nach Rolle im Zeitverlauf]

Dieser Bericht verfolgt die Aktivitäten von Einkaufsgruppen im Laufe der Zeit und hilft Ihnen zu verstehen, wie jüngste Kampagnen die Interaktion über verschiedene Rollen hinweg fördern. Verwenden Sie diese Informationen, um Ihre Marketing-Strategien zu optimieren und bestimmte Rollen effektiver anzusprechen.

![Dashboard „Rolleneinblicke“ - Interaktion nach Rolle im Zeitverlauf](assets/roles-insights-engagement-by-role-over-time.png){width="500" zoomable="yes"}

Jede Linie im Diagramm stellt eine Rolle dar. Bewegen Sie den Mauszeiger über einen Punkt auf der Linie, um die Details anzuzeigen, darunter:

* Datum
* Anzahl der Interaktionen

Um detailliertere Informationen anzuzeigen, klicken Sie auf das Menüsymbol **…** oben rechts.

## Interagieren mit den Daten

Um mit den Daten zu interagieren _verwenden Sie das Menü_&#x200B;**Mehr…**) oben rechts in jedem Diagramm.

### [!UICONTROL Drill-Through]

Wählen Sie für _[!UICONTROL Interaktion nach]_ die Option **[!UICONTROL Drill-Through]**, um eine detaillierte Analyse der Interaktionen nach Rolle und Lösungsinteresse durchzuführen.

Die auf das Dashboard angewendeten globalen Filter werden übernommen. Klicken Sie oben links auf _Filter_ ( ![Filtersymbol](../assets/do-not-localize/icon-filter.svg) ), um [die Attributfilter zu ändern](#filter-the-data) für die Drill-Through-Ansicht.

### [!UICONTROL Mehr anzeigen]

Wählen Sie **[!UICONTROL Mehr anzeigen]**, um erweiterte Daten und Einblicke anzuzeigen.

Das angezeigte Popup enthält ein Diagramm und eine Tabelle, die die Aufschlüsselung der Daten anzeigt.
