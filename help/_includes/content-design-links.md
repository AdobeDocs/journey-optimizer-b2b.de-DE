# Inhaltserstellung - Linktracking

Das Linktracking-Bedienfeld bietet eine zentrale Ansicht, in der Sie jede URL im Inhalt einfach finden können. Klicken Sie auf _[!UICONTROL Links]_-Symbol ( ![Link-Symbol anzeigen](../assets/do-not-localize/icon-links.svg) ) auf der linken Seite, um alle verknüpften URLs in Ihrem Inhalt anzuzeigen. Dies sind die Links innerhalb von Text-, Schaltflächen- oder Bildkomponenten.

Um das Tracking für einen aufgelisteten Inhaltslink zu definieren, klicken Sie auf das Symbol _Bearbeiten_ ( ![Bearbeiten-Symbol](../user/assets/do-not-localize/icon-edit.svg) ) und ändern Sie **[!UICONTROL Tracking-Typ]** oder **[!UICONTROL Label]**.
